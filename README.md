# CRM-lite (этап 1)

CRM-lite – это учебное веб-приложение для малого бизнеса. На первом этапе реализованы:
- Регистрация и вход пользователей (JWT).
- Расширенная модель пользователя (владелец / сотрудник).
- Создание компании (только владельцем, одна компания на пользователя).
- Создание склада (только владельцем, один склад на компанию).

## Технологии
- Python 3.10+
- Django 4.2
- Django REST Framework (DRF)
- djangorestframework-simplejwt (JWT аутентификация)
- drf-spectacular (Swagger документация)
- SQLite (по умолчанию) / PostgreSQL
